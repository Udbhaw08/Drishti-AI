Drishti Frontend
Drishti is an AI-powered surveillance dashboard for real-time monitoring and smart public safety alerts. This frontend lets operators view multiple camera feeds, respond to alerts, and access incident reports through a user-friendly interface.

Features
Live video from CCTV, drones, and bodycams

Instant AI alerts (weapons, suspicious behavior)

Heatmap and event history views

Easy incident reporting

Tech Stack
React.js

Tailwind CSS

Getting Started
Clone this repo:

bash
git clone https://github.com/yourusername/Drishti-AI.git
cd drishti-frontend
Install dependencies:

bash
npm install
Run the app:

bash
npm start
Usage
Log in and select your role (Admin/Operator)

Monitor live feeds, respond to alerts, and view history
